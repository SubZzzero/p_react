# Pizza App

Educational frontend project built with React, TypeScript and Redux Toolkit.

## Description

This is an educational single-page application created to practice modern frontend development with React and Redux Toolkit.

The application allows users to browse a list of pizzas, filter and sort them, search by name, add items to the cart and persist cart data in localStorage.

The project was developed for learning purposes and as part of a personal portfolio.

## Features

- Fetching pizza data from API
- Search functionality
- Category filtering
- Sorting
- Pagination
- Shopping cart
- Cart persistence with redux-persist
- Skeleton loading

## Tech Stack

- React 18
- TypeScript
- Redux Toolkit
- React Redux
- Redux Persist
- React Router DOM
- Axios
- React Scripts 
- CSS
- HTML
